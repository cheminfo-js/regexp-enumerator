enumerateRegExp

src/index.js

Function that generates all possible strings given a regular expression.

enumerateRegExp(regexp: (string | regexp), options: object): Array
Parameters
regexp ((string | regexp)) regular expression to use
options (object = {})
Name Description
options.universe regexp (default /[a-z]/) universe of letters allowed for the 'not' operator.
options.maxSize number (default 10) max string size of the generated output.
Returns
Array: generated string given the regular expression.
